I am a PhD candidate in Computer Science at Yale University, advised by Professor Lin Zhong. Currently, I am excited about AI for systems: using large language models to develop and transform systems software at scale. My recent work builds AI agentic pipelines that generate and refine Hardware Abstraction Layers (HALs) and transform Rust code to adopt compiler-checked design patterns, making it easier to generate correct systems code at scale.

More broadly, I work on correctness in systems software — building low-level systems we can trust. Rather than full formal verification, I lean on lightweight techniques such as the type system to enforce key system invariants with little overhead.

You can find my publications, my CV, and the open-source projects I contribute to. The best way to reach me is by email.
